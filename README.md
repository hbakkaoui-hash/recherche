# Recherche — Hassane Bakkaoui

Site vitrine regroupant mes travaux de recherche en **physique théorique** et en
**mathématiques**, ainsi que mes projets logiciels.

- **Physique** — programme « dualité d'échelle compactifiée » (Unités 1 à 5).
- **Mathématiques** — familles paramétriques de nombres premiers, problèmes d'Erdős,
  contributions en Lean.
- **Projets** — applications web.

## Structure

- `index.html`, `styles.css` — page et mise en forme.
- `data.js` — contenu (publications, projets). Le seul fichier à éditer pour ajouter du contenu.
- `render.js` — génération des cartes.
- `papers/` — versions PDF des articles.

Le site est statique (aucune compilation) et publié via **GitHub Pages**.
